# 📱 KosSmart App

## 📌 Deskripsi

KosSmart adalah aplikasi pencarian kos berbasis mobile menggunakan Flutter GetX yang menyediakan fitur pencarian, booking, pembayaran, dan review.

---

## 🚀 Fitur

* Login
* Pencarian kos
* Filter kos
* Detail kos
* Booking kos
* Pembayaran
* Review
* Tambah kos
